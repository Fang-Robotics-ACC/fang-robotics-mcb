Dependency Injection
====================

DI is a common pattern within programming to satisfy the Dependency Inversion 
principle of SOLID. It allows a great deal of code reuse without duplication. 
Here, we will address the basics, concerns, and the specifics of how and why 
Fang Robotics MCB utilizes it and adapts it slightly from traditional patterns.
